# 第十阶段：课程论文图表与结果素材说明

## 1. 素材范围

本阶段依据独立测试集的真实评估结果，统一生成课程论文所需图表。所有模型使用相同的测试集进行比较，避免将验证集最优结果与测试集结果混用。图表位于 `outputs/report_assets/figures`，数据表位于 `outputs/report_assets/tables`。

## 2. 可写入论文的核心结果

基线模型 YOLO26n@640 在测试集上的精确率、召回率、mAP@0.5 和 mAP@0.5:0.95 分别为 0.6944、0.7796、0.7711 和 0.5667。采用容量更大的 YOLO26s 并保持 640×640 输入后，四项指标分别达到 0.7373、0.7926、0.8221 和 0.6065。其中 mAP@0.5 提高 5.10 个百分点，相对增幅约 6.61%；mAP@0.5:0.95 提高 3.99 个百分点，相对增幅约 7.04%。

提高输入分辨率的 YOLO26n@960 获得最高的总体 mAP@0.5（0.8257）和 mAP@0.5:0.95（0.6083），但它的阅读、抬头听讲、转头和举手 AP@0.5 均低于 YOLO26s@640，且训练耗时由 278.45 s 增至 532.83 s。综合关键行为类别、标准输入尺寸及部署便利性，系统最终选择 YOLO26s@640，而不是仅依据总体 mAP 的微小差异选择 YOLO26n@960。

逐类别结果显示，最终模型相对基线在阅读、转头和举手类别上的 AP@0.5 分别提高 10.92、11.79 和 12.74 个百分点，说明增加模型容量能够改善外观相近行为的特征表达。站立类别下降 11.31 个百分点，讨论类别下降 0.41 个百分点。站立样本数量相对有限，测试结果容易受到少量误检和漏检影响，这一结果应作为系统局限性如实说明，而不应隐藏。

## 4. 表格用途建议

`model_comparison.csv` 可转写为“不同实验方案综合性能对比表”；`per_class_metrics.csv` 可转写为“各行为类别检测结果表”；`dataset_split_summary.json` 保存数据划分的原始统计，后续排版阶段将转换为正式三线表。

## 5. 结果表述边界

系统输出的是逐帧行为检测框及逐帧类别计数，并未实现跨帧身份跟踪。因此报告中可以表述为“课堂行为检测与统计”，不能把累计检测次数描述为去重后的学生人数，也不能直接将检测结果等同于教学质量评价。
