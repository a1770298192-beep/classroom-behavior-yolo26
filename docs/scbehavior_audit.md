## 相似场景风险

使用 64 位差异哈希进行初步筛查后发现，汉明距离为 0 的图像对有 194 对，其中 36 对跨越上游 train/val 划分；距离不超过 2 的图像对有 3111 对，其中 541 对跨划分。这不代表文件完全重复（SHA-256 审计未发现完全重复图片），但说明数据包含大量视觉上高度相似的课堂场景，上游随机划分存在场景泄漏风险。

因此后续不能直接沿用上游 360/40 划分作为最终实验划分。阶段4需要依据视觉场景聚类或来源分组重新建立 train/val/test，并保存可复现清单。
