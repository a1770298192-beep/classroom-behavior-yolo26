# 数据使用与隐私原则

## 数据来源记录

每个实际使用的数据集必须记录：来源链接、下载日期、版本或提交号、许可或使用声明、引用论文、原始类别顺序和本项目所做的修改。

## 数据处理原则

- `data/raw` 中的原始文件不做覆盖修改。
- 发现损坏图片、异常标注、重复样本或越界框时，记录原因后在处理副本中修正或排除。
- 若图片来自视频连续帧，应按视频或场景划分数据集，不能随机把相邻帧分到训练集和测试集。
- 数据集没有明确许可时，仅限课程学习和本地实验，不重新分发原始数据。

## 隐私原则

- 不做人脸识别，不保存学生身份信息。
- 输出课堂整体行为统计，不制作个人排名或个人学习评价。
- 阅读、低头、转头等可观察行为不直接等同于注意力或学习态度。
- 若后续采集本校课堂素材，应获得授权，并优先采用人脸模糊、远景拍摄或模拟课堂数据。
